== Installation ==

There are 3 ways to install this plugin:

= 1. The Easy Method =

1. In your Admin, go to menu Plugins > Add
2. Search for CP Analytics Pro
3. Click to install
4. Activate the plugin
5. A new option under Settings menu named `CP Analytics Pro` will appear in your Admin

= 2. Install Through Zip File =

1. Download the plugin (.zip file) on the right column of this page
2. In your Admin, go to menu Plugins > Add
3. Select the tab "Upload"
4. Upload the .zip file you just downloaded
5. Activate the plugin
6. A new option under Settings menu named `CP Analytics Pro` will appear in your Admin

= 3. Through FTP =

1. Upload `CP Analytics Pro` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A new option under Settings menu named `CP Analytics Pro` will appear in your Admin

== Usage ==
After installing the CP Analytics Pro Plugin, You need to click on 'Settings' to activate your plugin. The steps include:

1. Click on the 'Authentication' button
2. Log in to the Gmail Account of your website
3. When you open the link of CP Analytics Pro , you will see a secret access code. You need to copy this access code and paste it on the 'Your access code'.
4. Click on save changes. You will have to save twice for the database and server connect.
5. Select your domain name, click on save changes.
You will get the analytics report once you complete these steps.
